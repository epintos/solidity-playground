# Solana GIFs portal project

Create a local keypair:

```shell
  solana-keygen new
  solana address
```

Change to dev net:
```shell
  solana config set --url devnet
  solana config get
  solana airdrop 5
  solana balance
```

You can also use the [faucet](https://solfaucet.com/) if the local airdrop doesn't work
